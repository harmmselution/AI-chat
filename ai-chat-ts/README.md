# AI Chat App

A chat application built with **React**, **TypeScript**, **Vite**, and **SCSS Modules**, powered by the **Hugging Face Chat Completions API**.

It supports:

- streaming AI responses
- stop streaming
- persistent chat history with `localStorage`
- clear chat history

---

## Tech Stack

- React
- TypeScript
- Vite
- SCSS Modules
- react-hot-toast
- Hugging Face Inference API

---

## Project Setup

### 1. Clone the project

```bash
git clone <your-repo-url>
cd ai-chat-ts

```

### 2. Install dependencies

```bash
yarn

```
### 3. Create environment variables
VITE_HF_API_KEY=your_huggingface_api_key_here

Running the Project
yarn dev
http://localhost:5173
